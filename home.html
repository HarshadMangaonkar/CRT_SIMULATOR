About the Project

A Cathode Ray Tube (CRT) oscilloscope helps students understand signal visualization, electron emission, focusing, and deflection. Since real CRTs are outdated and fragile, this virtual simulator provides a safe and interactive learning experience using web technology.

Main Features

• Real-time electron beam animation
• Adjustable beam speed & intensity
• Magnetic & electric deflection controls
• Realistic physics-based simulation
• Built using HTML, CSS & JavaScript
