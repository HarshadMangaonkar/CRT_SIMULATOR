About the project

A Cathode Ray Tube (CRT) oscilloscope helps students understand signal visualization, electron emission, focusing, and deflection. Since real CRTs are outdated, fragile, and rarely available, students often learn only the theory. A Virtual CRT Oscilloscope Simulator visually demonstrates beam movement, deflection, and waveform formation using web technologies, making learning interactive, safe, and easier without real hardware.

Main Features

• Real-time electron beam animation
• Adjustable beam speed & intensity
• Magnetic & electric deflection controls
• Realistic physics-based simulation
• Built using HTML, CSS & JavaScript
